# Udon Thani International Horticultural Expo 2026

เว็บไซต์ต้นแบบแบบหน้าเดียวสำหรับโจทย์ Work1 — รองรับ 3 ภาษา (ไทย / English / 中文) และทำงานได้ทั้ง Desktop, Tablet และ Mobile

## วิธีเปิด

เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ได้ทันที หรือเปิดโฟลเดอร์นี้ด้วย VS Code แล้วใช้ Live Server

## ฟังก์ชัน

- สลับภาษาไทย อังกฤษ และจีนทั้งหน้า
- โหมดมืด / สว่าง
- Countdown วันเปิดงาน
- Cinematic parallax, animated light rays, water shimmer, floating leaves และพื้นหลังเคลื่อนไหว
- แผนที่โซนแบบโต้ตอบ และแผนที่สถานที่จริงที่เลื่อน/ซูมได้
- Timeline และตารางกิจกรรมที่กดเปลี่ยนได้
- Modal เรื่องราวและแบบฟอร์มสาธิต
- Responsive และรองรับ `prefers-reduced-motion`
- ภาพ Hero และสวนเด่นสร้างขึ้นเฉพาะสำหรับงานนี้

> หมายเหตุ: รายชื่อประเทศ กิจกรรม ข่าวสาร และแบบฟอร์มบัตรบางส่วนเป็นข้อมูลตัวอย่างสำหรับการนำเสนอผลงานออกแบบ ไม่ใช่ข้อมูลจำหน่ายบัตรอย่างเป็นทางการ
